
# 💊 Medicine Info App

**Description:**
Interactive web app providing detailed medicine information, health tips, and pathways.

**Features:**

* User login and registration
* Search and explore medicines
* Dosage combinations and alternatives
* Side effects for each medicine
* Health tips with search/filter
* Medicine classification cards
* Medicine pathway flow charts
* Quick info and reference sections
* Easy-to-use sidebar navigation

**Technologies Used:**

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* LocalStorage for login/authentication

**Usage:**

1. Open `index.html` in a browser.
2. Register or login.
3. Navigate using the sidebar to explore medicines, health tips, side effects, and pathways.
4. Use search fields to filter medicines or tips.
